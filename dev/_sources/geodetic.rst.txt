Geodetic
========

Module: ``sasktran.geodetic``

Internally SASKTRAN uses the `World Geodetic System (WGS48) <https://en.wikipedia.org/wiki/World_Geodetic_System>`_.
This module provides some helpers for working with geodetic coordinates.
